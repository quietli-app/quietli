Brain Blip clean local build

1. Copy .env.local.example to .env.local and fill in your Supabase values.
2. Run the SQL in supabase_schema.sql inside the Supabase SQL editor.
3. Create a public bucket named avatars.
4. Run npm install
5. Run npm run dev

Notes:
- This package does not include node_modules.
- Keep your own .env.local private.
- If you already have a working Supabase project, you can reuse it.
